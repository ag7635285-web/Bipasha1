# Amar Crash Website

এই প্রজেক্টটি একটি ছোট ওয়েবসাইট যা আপনার ক্রাশ বা প্রিয় বন্ধুর জন্য বানানো হয়েছে।

## কী আছে

- `index.html` — প্রধান পেজ
- `styles.css` — সাইটের স্টাইল
- `manifest.json` — PWA অ্যাপ ফিজার
- `sw.js` — সার্ভিস ওয়ার্কার
- `icon.svg` — অ্যাপ আইকন

## GitHub Pages এ পাবলিশ করার জন্য

1. একটি GitHub রিপোজিটরি তৈরি করুন
2. এই ফোল্ডারটি রিপোজিটরিতে পুশ করুন
3. GitHub Settings > Pages থেকে `main` branch এবং `root` নির্বাচন করুন
4. আপনার সাইট পাবলিক URL পেয়ে যাবে
